## Summary

Phase 21 closes the critical architectural gap identified at Phase 20: the
intelligence stack built in Phases 16ÔÇô20 (`AutonomyLoop`, `IntelligenceRouter`,
`CritiqueSignalBuffer`, `RoutedDecisionTelemetry`) was fully isolated from the
production `EvolutionLoop.run_epoch()` execution path. It was never called.

v7.1.0 wires `AutonomyLoop.run()` as **Phase 5g** inside `run_epoch()`, feeding
it live epoch signals (mutation score, debt score, fitness trend, pass rate,
lineage health) and surfacing the intelligence decision in `EpochResult`.
