## Summary

Phase 75 completes the full Seed Lifecycle Pipeline (Phases 71‚Äì75). `inject_seed_proposal_into_context()` merges a seed-derived `ProposalRequest` into a CEL epoch context dict. `SeedCELInjectionEvent` is written to the lineage ledger before return. ### CEL Step 4 Wiring

`LiveWiredCEL.run_epoch` ‚Üí Step 4 now calls `resolve_step4_request(state_context)`:
- If `seed_proposal_request` key present: returns injected `ProposalRequest`
- Fallback: returns `default_proposal_request()` (no behavioural change for un-seeded epochs)

### New API Endpoint

`POST /seeds/promoted/{seed_id}/inject` ‚Äî returns ready `epoch_context` dict with injected proposal
