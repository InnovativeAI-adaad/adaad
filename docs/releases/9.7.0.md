# ADAAD v9.7.0 — Phase 72: Seed Promotion Queue + Graduation UI

**Released:** 2026-03-14
**Phase:** 72 — Seed Promotion Queue + Graduation UI
**Requires:** v9.6.0 (Phase 71 complete)
**Seed Lifecycle Phase:** 2 of 5

---

## Summary

Phase 72 introduces the advisory Seed Promotion Queue. Graduated seeds (expansion_score ≥ 0.85) enter a FIFO queue — never automatically acted upon without human review (`SEED-PROMO-HUMAN-0`). The Aponi dashboard gains a gold graduation toast, live seeds list with graduated count badge, and an Oracle Query History card showing the last 20 ledger records.

---

## What Shipped

### SeedPromotionQueue

| Capability | Invariant | Detail |
|---|---|---|
| Advisory FIFO queue | `SEED-PROMO-0` | Graduated seeds enqueued; never auto-promoted without human decision |
| Idempotent enqueue | `SEED-PROMO-IDEM-0` | Duplicate seed_id is a no-op |
| Human advisory notice | `SEED-PROMO-HUMAN-0` | Every queue response carries explicit human-review advisory |
| FIFO ordering | `SEED-PROMO-ORDER-0` | Graduation order preserved; oldest-first on dequeue |

### New API Endpoint

`GET /innovations/seeds/promoted` — queue depth, entries, and `SEED-PROMO-HUMAN-0` advisory notice

### Aponi Additions

| Element | Detail |
|---|---|

| Seeds panel header badge | Live graduated count displayed in Seeds panel header |
| Oracle Query History card | Last 20 `OracleLedger` records rendered newest-first; query type, trajectory score, timestamp |
