# ADAAD v9.5.0 — Phase 70: WebSocket Live Epoch Feed

**Released:** 2026-03-14
**Phase:** 70 — WebSocket Live Epoch Feed
**Requires:** v9.4.0 (Phase 69 complete)

---

## Summary

Phase 70 delivers real-time epoch observability over WebSocket. An async `InnovationsEventBus` with 8 typed emit helpers replaces ad-hoc event dispatch throughout the CEL pipeline. The `/ws/events` endpoint is upgraded to a persistent stream. Aponi gains live epoch progress visualization, personality badges, arc prepend, and reflection/seed/governance plugin toasts.

---

## What Shipped

### InnovationsEventBus

| Capability | Invariant | Detail |
|---|---|---|
| Async fan-out dispatch | `IBUS-FANOUT-0` | All registered handlers notified; no subscriber blocks others |
| Fail-safe emit | `IBUS-FAILSAFE-0` | Handler exceptions caught and logged; bus never crashes epoch |
| Deterministic frame shape | `IBUS-DETERM-0` | Equal event inputs → identical emitted frame |
| Thread safety | `IBUS-THREAD-0` | Handler registration and emit are lock-protected |

### 8 Typed Emit Helpers

`emit_epoch_start` · `emit_epoch_end` · `emit_story_arc` · `emit_cel_step` · `emit_personality` · `emit_reflection` · `emit_seed_frame` · `emit_gplugin_result`

### CEL Wiring

- `LiveWiredCEL.run_epoch` emits `epoch_start` / `epoch_end` frames
- Step 14 emits `story_arc` + `cel_step`
- Personality and reflection injection points emit typed frames

### Aponi Live UI

| Element | Detail |
|---|---|
| Tri-color epoch progress bar | Blue/teal/amber segments advance with CEL step count |
| Personality header badge | Active agent persona displayed live during epoch |
| Live arc prepend | Story arc frames prepend to Story Mode panel as they arrive |
| Reflection / seed / G-plugin toasts | Typed WS frames trigger panel-specific toast notifications |
