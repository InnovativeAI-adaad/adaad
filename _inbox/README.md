# _inbox

This folder holds working documents, media assets, and proposals that are not part of the canonical ADAAD codebase.

| Subfolder | Contents |
|---|---|
| `media/` | Images, audio, banner assets, AI-generated visuals |
| `reports/` | Deep-dive audits, analysis documents, CSV exports |
| `proposals/` | Engineering proposals, roadmap docs, HTML presentations |

These files are preserved for reference but excluded from production builds and CI gating.
